=== Plugin Installation===

How to install this plugin
1)Unzip and upload the php file to your wordpress plugin directory.
2)Activate the plugin from your wordpress plugin panel.
3)appearence-->widget-->drag the widget in your sidebar panel

=== Plugin settings===
Setting:
In wordpress dashboard Go to settings-->sidebar login
